# phpinfo.wp
This is a troubleshooting purpose WordPress plugin to view your PHP Info.
Look into __PHP Info (WP)__ link in Admin menus, right after you installed it.


## Download
 - Slug: php-info-wp
 - [SVN](https://plugins.svn.wordpress.org/php-info-wp/)
